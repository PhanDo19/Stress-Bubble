# Stress Bubble

Stress Bubble is a Chrome Extension mini-game built for fast stress-relief sessions.

## Features

- New Tab game experience
- Toolbar popup quick-play mode
- Three modes: `classic`, `rage`, `zen`
- Persistent best scores, coins, streaks, shop unlocks, achievements, and settings

## Load Unpacked

1. Open `chrome://extensions`
2. Enable `Developer mode`
3. Click `Load unpacked`
4. Select the project root folder

## Test Flows

- Open a new tab and verify the home screen appears
- Start a run and verify result screen, replay, and persistence
- Click the extension icon and verify popup quick-play works
- Verify settings and reset data behave correctly

## Known Limitation

- Chrome does not apply New Tab overrides in Incognito windows

## Privacy

The extension stores gameplay progress and settings locally in browser storage.

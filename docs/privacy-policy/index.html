Stress Bubble

Privacy Policy

Stress Bubble stores limited gameplay data locally in your browser so the extension can remember progress and settings.

Data stored locally

  • best scores by mode
  • coins
  • streak progress
  • daily challenge state
  • shop unlocks and selected skin
  • gameplay settings
  • tutorial completion state
  • unlocked achievements

How data is used

  • to restore your game state between sessions
  • to keep your settings and progression available on the same browser profile
  • to power gameplay systems such as streaks, challenges, and rewards

Data sharing

  • Stress Bubble does not send this data to a remote server
  • Stress Bubble does not sell or share this data with third parties
  • Stress Bubble does not use this data for advertising

Storage location

Data is stored in chrome.storage.local when available, with local browser fallback storage used only outside the extension environment.
